# blih-gui
A cross-platform application to manage Blih.

#### Technologies
* Electron
* AngularJS
* Bootstrap Twitter
* GulpJS
* Blih API

#### Development

```sh
cd app
npm install
cd ..
npm install
```

Run with watching files

```sh
gulp serve
```

#### Production

Create a new directory with minified files and clean developpement tools

```sh
gulp build
```
